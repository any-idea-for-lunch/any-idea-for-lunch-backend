개발,협업 경험이 많지 않은 상태에서 프로젝트를 시작하여 초기에는 GitHub 연동, 프로젝트 설정 등 개발 환경을 구성하는 과정부터 팀원들의 도움을 받았습니다.  

팀원들의 도움으로 GitHub 협업 방식과 프로젝트 구조를 이해할 수 있었고, 단순히 코드를 작성하는 것을 넘어 팀 단위 개발 흐름을 경험할 수 있었습니다.

이후 기획과 디자인을 바탕으로 화면 단위로 UI를 하나씩 구현해 나가며 디자인 의도를 코드로 옮기는 과정에 집중했습니다.  
특히 사용자 흐름을 고려하여 메인환경부터 결과 확인까지의 동작이 자연스럽게 진행되게 구성했습니다.

개발 중 이해가 부족한 부분이나 막히는 문제에 대해서는 AI 도구를 활용하여 문제의 원인을 파악하고 해결 방안을 검토하였습니다.  
이를 통해 단순한 코드 복사가 아닌 ‘왜 이렇게 동작하는지’를 이해하려고 노력했습니다.
